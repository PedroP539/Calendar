# Assets - Instruções

## Imagens Necessárias

### Folha Base
- **Caminho**: `folha_base.png`
- **Dimensões**: 335mm × 835mm
- **Formato**: PNG

### Macetes
Cada macete precisa de 12 imagens (uma por mês):

#### Macete 1
- **Pasta**: `macete1/`
- **Ficheiros**: `macete1.png` a `macete12.png`
- **Dimensões**: 335mm × 165mm
- **Formato**: PNG

#### Macete 2
- **Pasta**: `macete2/`
- **Ficheiros**: `macete1.png` a `macete12.png`
- **Dimensões**: 335mm × 165mm
- **Formato**: PNG

#### Macete 3
- **Pasta**: `macete3/`
- **Ficheiros**: `macete1.png` a `macete12.png`
- **Dimensões**: 335mm × 165mm
- **Formato**: PNG

## Nomenclatura

Os ficheiros devem ser nomeados com o prefixo "macete" seguido do número:
- `macete1.png` = Janeiro
- `macete2.png` = Fevereiro
- ...
- `macete12.png` = Dezembro

## Notas

- As imagens são carregadas automaticamente
- Basta substituir os ficheiros para atualizar
- Não é necessário editar código
